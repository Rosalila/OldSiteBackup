This project is aimed to provide an online experience to horizontal shooting games (also known as STG, shoot'em up or shmup). This project is currently on alpha but you can already try to defeat the Feminist Kill Bot in Normal and Hard mode.

Linux dependencies and requirements

Download the correct .zip file, extract it and run the shell script as super user this way:
$ sudo sh install_dependencies.sh

  • Apt Linux 32 bits (Debian)
  • Apt Linux 64 bits (Debian)
  • Yum  Linux 32 bits (Fedora)
  • Yum Linux 64 bits (Fedora)

You will also need Java Runtime Enviroment (tested on 1.6 and 1.7).
