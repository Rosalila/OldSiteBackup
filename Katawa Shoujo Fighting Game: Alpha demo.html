Bug/Issue report

Last year, when we released Katawa Shoujo Divekick. There were some engine issues, we re-wrote the engine and hopefully most of them are now solved. If you find a bug/issue please report it at:
http://rosalilastudio.com/index.php/issuereport

This is a great way to contribute to the project.

Current state of the project

Right now the development will be slow because Rosalila is going to a crowd founding campaign for Maximum Overkill which is using the same engine for KSFG. The features added to the Maximum Overkill can be added to KSFG, such as story mode,  visual effects, Ouya support and engine optimizations. You can keep the track of Maximum Overkill here.
